# Agent 任務簡報模板

開始新任務時，可以用這份模板描述需求。

```txt
Project: ACG AI Picture Character LoRA
Task stage: <collection | curation | training | generation | evaluation>
Work title: <title>
Character name: <name>
Aliases: <optional>
Reference URLs: <optional>
Local files available: <yes/no>
Goal: <what needs to be produced>
Constraints: local experiment only, do not publish outputs
Deliverable: <file or result to create>
```

## 填寫說明

| 欄位 | 意義 |
|---|---|
| `Project` | 專案名稱 |
| `Task stage` | 目前任務階段 |
| `Work title` | 作品名稱 |
| `Character name` | 角色名稱 |
| `Aliases` | 角色別名，可省略 |
| `Reference URLs` | 參考網址，可省略 |
| `Local files available` | 是否已有本機圖片 |
| `Goal` | 這次要達成什麼 |
| `Constraints` | 限制條件 |
| `Deliverable` | 這次應產出的檔案或結果 |

## 建議預設值

- 一次只處理一個角色。
- 一次只要求一個明確交付物。
- 如果有模糊處，請在輸出筆記中寫清楚使用的假設。
