# Agent Task Brief

Use this as a prompt skeleton when starting a new task.

```txt
Project: ACG AI Picture Character LoRA
Task stage: <collection | curation | training | generation | evaluation>
Work title: <title>
Character name: <name>
Aliases: <optional>
Reference URLs: <optional>
Local files available: <yes/no>
Goal: <what needs to be produced>
Constraints: local experiment only, do not publish outputs
Deliverable: <file or result to create>
```

## Helpful Defaults

- Start with one character at a time.
- Prefer one clear deliverable per task.
- If something is ambiguous, write the assumption in the output notes.
